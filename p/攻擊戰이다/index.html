攻擊戰 ( 공격전 ) 이다

赤 ( 붉 ) 은 旗 ( 기 ) 　 上 ( 추 ) 켜 持 ( 들 ) 고　 進擊 ( 진격 ) 해 行 ( 가 ) ᄂ다,

총대앞세돌격가 銃隊를 前立우고 突擊해行ᄂ다;

일심 천만대오이끌가 一心의 千萬隊伍 引고行는,

그 모습 선군기치 其 面影은 先軍旗幟다.
