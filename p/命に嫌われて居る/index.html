命に嫌われて居る。

いのちにきらわれている。

1 minute read

初音ミク

作詞者 カンザキイオリ

作曲者 カンザキイオリ

リリース 2017年8月6日

命に嫌われて居る。

「死にたいなんて言うなよ。

諦め無いで生きろよ。」

其んな歌が正しいなんて

馬鹿げてるよな。

実際自分は死んでも良くて

周りが死んだら悲しくて

「其れが嫌だから」って言う

エゴなんです。

他人が生きても何うでも良くて

誰かを嫌う事もファッションで

其れでも「平和に生きよう」

なんて素敵な事でしょう。

画面の先では誰かが死んで

其れを嘆いて誰かが歌って

其れに感化された少年が

ナイフを持って走った。

僕等は命に嫌われて居る。

価値観もエゴも押し付けて

何時も誰かを殺したい歌を

簡単に電波で流した。

僕等は命に嫌われて居る。

軽々しく死にたいだとか

軽々しく命を見てる

僕等は命に嫌われて居る。

お金が無いので今日も

一日中惰眠を謳歌する

生きる意味なんて見出せず、

無駄を自覚して息をする。

寂しいなんて言葉で

此の傷が表せていい物か

其んな意地計り抱え

今日も一人ベッドに眠る

少年だった僕達は

何時か青年に変わってく。

年老いて何時か 枯 ( か ) れ 葉 ( は ) の様に

誰にも知られず 朽 ( く ) ちて 行 ( い ) く。

不死身 ( ふじみ ) の 身体 ( からだ ) を手に入れて、

一生死なずに生きて 行 ( い ) く。

其んなSFを妄想してる

自分が死んでも何うでも良くて

其れでも周りに生きて欲しくて

矛盾を抱えて生きてくなんて

怒られて仕舞う。

「正しい物は正しく居為さい。」

「死にたく無いなら生きて居為さい。」

悲しく成るなら其れでもいいなら

ずっと一人で笑えよ。

僕等は命に嫌われて居る。

幸福の意味すら分からず、

産まれた環境計り憎んで

簡単に過去計り呪う。

僕等は命に嫌われて居る。

さよなら計りが好き過ぎて

本当の別れ等知らない

僕等は命に嫌われて居る。

幸福も別れも愛情も友情も

滑 ( こっ ) 稽 ( けい ) な夢の 戯 ( たわむ ) れで

全部カネで買える 代 ( しろ ) 物 ( もの ) 。

明日死んで仕舞うかも知れ無い。

全て無駄になるかもしれない。

朝も夜も春も秋も

変わらず誰かが何処かで死ぬ。

夢も明日も何もいら無い。

君が生きて居たなら其れでいい。

其うだ。本当は其う言う事が歌いたい。

命に嫌われて居る。

結局何時かは死んで行く。

君だって僕だって何時かは枯れ葉に様に朽ちてく。

其れでも僕等は必死に生きて

命を必死に抱えて生きて

殺して 足掻 ( あが ) いて笑って抱えて

生きて、生きて、生きて、生きて、

生きろ。
